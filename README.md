# PocketMind 🤖

An AI-powered chat app built with React Native and Expo.
Chat with AI and access your conversations offline, anytime.

## Features

- 💬 AI chat powered by OpenRouter API
- 💾 Offline storage — chats saved on device
- 📋 Multiple chat sessions
- 🗑️ Delete chats with long press
- 🌙 Dark theme UI

## Tech Stack

- React Native
- Expo SDK 54
- React Navigation
- AsyncStorage
- OpenRouter API

## Setup

1. Clone the repo
```
git clone https://github.com/kartik5151/PocketMind.git
```

2. Install dependencies
```
npm install
```

3. Add your API key — rename `src/api/gemini.example.js` to `src/api/gemini.js` and add your OpenRouter API key

4. Run the app
```
npx expo start
```
